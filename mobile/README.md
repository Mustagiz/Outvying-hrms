# HRMS Mobile App

This directory is reserved for the React Native (Expo) application.

## Quick Start (Pre-requisites)
To initialize the mobile app, run:

```bash
npx create-expo-app@latest .
```

*Note: This command is interactive, so it must be run manually by the developer.*

## Planned Features
- Clock In/Out with Geofencing
- Payslip View
- Leave Requests
- Push Notifications
